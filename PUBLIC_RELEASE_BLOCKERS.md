Author approval to publish this public reproducibility package was provided on 2026-05-15. The package is scoped to code, runbooks, source registries, derived tables and generated figures.

The package intentionally excludes:

1. Raw third-party datasets and downloaded archives.
2. Active submission manuscripts and reviewer-facing drafts.
3. Private logs, credentials and local environment files.
4. Journal cover letters, reviewer-response drafts and internal round logs.
5. Any files whose redistribution rights are unclear.
